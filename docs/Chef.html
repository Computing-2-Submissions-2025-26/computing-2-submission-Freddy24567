Chef

Namespace

Chef

Description:

  • Chef.js provides computer opponents ("chefs") for Boxed Up. A computer chef examines a game using only the public BoxedUp API and proposes a ply for the player to move, at one of three skill levels. Chefs are deterministic and pure: the same game always gets the same proposal.

(static) choose_ply(level, game) → {Chef.Ply|undefined}

Description:

  • Proposes a ply for the player to move, at the given skill level. Returns undefined if the game has already ended. The proposal is always a legal ply; pass it to BoxedUp.place to make the move.
